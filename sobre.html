Programação para Internet

Sobre a disciplina:

A disciplina Programação para Internet é uma das disciplinas do curso de Sistemas de Informação ofertado no 4º período pela Universidade Federal de Uberlândia. Atualmente a disciplina é ministrada pelo Prof. Dr. Rafal D. Araújo.

Objetivo:

Introduzir o paradigma da programação para a Internet, que possui uma lógica de construção de programas substancialmente diferente das demais; Conceituar as arquiteturas de 2, 3 e n camadas, apresentando as diferenças e, principalmente, as vantagens que elas oferecem; Qualificar profissionais para o desenvolvimento de aplicações voltadas para a Internet/Intranet e que sejam disponibilizadas através de um browser; Apresentar padrões de projeto para melhor estruturação dos sistemas na web; Apresentar as tecnologias mais empregadas no desenvolvimento de sistemas para web, inclusive com o uso de frameworks; Introduzir conceitos de segurança associados ao uso de sistemas na web.
